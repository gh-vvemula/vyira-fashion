# Vyira Fashion

Performance-focused fashion storefront for a contemporary men and women brand. The catalog intentionally excludes saree, ethnic, and traditional wear, and focuses on scalable category-driven merchandising.

## Stack

- Next.js App Router
- TypeScript
- CSS with reusable design tokens
- Static collection and product data for easy extension

## Run locally

```bash
npm install
npm run dev
```

## Current sections

- Home landing page
- Women page
- Men page
- New In page
- Dynamic collection pages

## Notes

- The structure is organized for future CMS, API, or commerce integration.
- Most UI is server-rendered for strong performance defaults.
